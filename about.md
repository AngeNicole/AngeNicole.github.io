# About Book Hub

Book Hub is a personal project created to celebrate the joy of reading. It started as a full-stack web application where users could browse books, write reviews, track their reading progress, and discover new titles based on their interests. This static site brings that same passion for books to a simple Jekyll-powered website hosted on GitHub Pages.

---

## About Me

I'm a student and developer who loves both technology and literature. I believe that sharing book recommendations and honest reviews helps build a community of curious readers. Book Hub is my way of combining these two passions.

---

## The Original Book Hub App

The original Book Hub is a React + Node.js web application featuring:

- **Book browsing** with filters by genre, author, rating, and publication date
- **User accounts** with personal reading lists, wishlists, and liked books
- **Reviews and ratings** where readers share their honest opinions
- **Reading progress tracking** so you can pick up where you left off
- **Admin dashboard** for managing books, genres, users, and reviews
- **Genre-based recommendations** tailored to each reader's interests

This Jekyll site takes the content and spirit of that application and presents it as a clean, lightweight static website — no database, no login, just books and honest opinions.
