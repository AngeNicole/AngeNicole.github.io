## Suggest a Book

If you have read something great and want to see it featured on Book Hub, feel free to open an issue on the GitHub repository with the following details:

1. **Book title** and **author**
2. **Genre**
3. A short note about **why you recommend it**

I review all suggestions and add the best ones to the site.

## About This Project

This website was built as part of a Coursera assignment on GitHub Pages and Jekyll. It is inspired by my full-stack Book Hub application, which allows users to browse, review, and track their reading progress.

- **Static Site Generator:** Jekyll
- **Hosting:** GitHub Pages
- **Theme:** Cayman (a GitHub Pages supported theme)
- **Content:** Written in Markdown
